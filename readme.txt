Installation

1. In tre root folder run 'npm install'
2. Run 'gulp'
3. Open localhost in browser (http://127.0.0.1:3000/home)


Testing

1. Login like differend users  
	User A - head of the accounting department (login: user_a, password: admin)
	User B - employee of the accounting department (login: user_b, password: admin)
	User C - employee of the HR department (login: user_c, password: admin)
2. Press 'Start new process' button
3. choose 'Innovation management process' and enter the title of the process
4. Press 'Create and start first step' button
5. For different users uoy will see different combination of recommended actions for the current step and recommended next step
